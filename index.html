Welcome to the Storefront API.

Please use the following routes are supported:

User routes

  • Authenticate User [POST]
  • Create new Users [POST]
  • Get User by Id [GET]
  • get All Users [GET] [token required]
  • Update User [PUT]
  • Delete User [DELETE]

Product routes

  • get All Product [GET] [token required]
  • get Product by id [GET] [token required]
  • create new Product [POST] [token required]
  • Update Product [PUT] [token required]
  • Delete Product [Delete] [token required]

Order routes

  • get All Order [GET][token required]
  • get Order by id [GET][token required]
  • create new Order [POST][token required]
  • update Order by Id [PUT][token required]
  • delete Order by Id [DELETE] [token required]
